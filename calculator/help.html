Key Binds
s → sin()
c → cos()
t → tan()
ctrl + s → arcsin()
ctrl + c → arccos()
ctrl + t → arctan()
p → π
e → ℯ
q → √
slash or d → ÷
^ → ^()
x → ×
enter → =
Key Explanations
entry→ get the previous entries that have been entered
rnd:→ changing the rounding in decimal places. Rnd:# (# is the number)
rad/deg → change between radians and degrees
mod → modulus
rand → random number from 0-1
